# 綠界 全方位(All In One)金流介接Java版
---

## 1. 介紹

  - 綠界對於有收款需求的會員，提供完整的交易介接API，並有多種收款方式(All In One)可選擇使用。 本套件為Java版，可使用作建立訂單，接受付款通知，查詢訂單等金流交易的應用。
  - 收款方式清單：
    - 信用卡(一次付清、分期付款、定期定額)
    - 網路ATM
    - ATM櫃員機
    - 超商代碼
    - Apple Pay
    - Android Pay


## 2. 安裝環境
  - _jdk 1.6以上_
  - SDK元件限制在伺服器上使用，嚴禁使用於手機環境，已避免商家金鑰資訊外流
